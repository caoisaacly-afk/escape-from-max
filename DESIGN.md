# Escape from Max - Design Document

## 1. 做给谁（目标用户）
喜欢恐怖生存类小游戏的休闲玩家。想在浏览器中体验被怪物追杀的紧张刺激感，无需下载安装，打开即玩。适合快速解压、朋友间互相挑战生存时长。

## 2. 解决什么问题
- 市面上缺少轻量、免费、浏览器即开的恐怖追逐游戏
- 大多数恐怖游戏需要下载、配置要求高
- 需要一个快速上手、紧张刺激的休闲小游戏
- 在别墅环境中加入追逐玩法，既有探索感又有压迫感

## 3. 核心功能列表

### 3.1 别墅地图
- 俯视视角 2D 别墅，包含多个房间：
  - 客厅（大房间，有沙发、茶几等家具）
  - 餐厅（有餐桌、椅子）
  - 厨房（有灶台、冰箱）
  - 走廊（连接各个房间）
  - 主卧（有床、衣柜）
  - 浴室
  - 书房（有书桌、书架）
- 房间之间通过门洞连接
- 家具作为障碍物，玩家和怪物都需要绕行
- 地毯/地垫（深棕色花纹），分布在各房间中增强别墅氛围

### 3.2 玩家操控
- WASD 移动，支持斜向移动
- 玩家比 Max 稍快，可以通过走位甩开
- 碰撞检测：不能穿过墙壁和家具
- 支持手机触屏（触摸移动 + 虚拟摇杆）
- 被动道具：被 Max 碰到后短暂无敌（2秒，闪烁）

### 3.3 怪物 Max
- 红色眼睛，在远处会变暗
- 使用 BFS 寻路算法追踪玩家，会绕过墙壁和家具
- 速度接近玩家，随时间缓慢增加
- 越靠近玩家，屏幕震动越强、心跳效果越明显
- 3 条命系统：被 Max 碰到扣 1 条命，3 条命耗尽才死亡
- 可被击晕：玩家持有刀时自动反击，Max 被击晕 3 秒（变灰 + 旋转星星）
- 隐身状态：玩家隐身时 Max 失去目标，随机游荡

### 3.4 光照系统
- 别墅整体可见（深色调），边缘有渐变暗角
- 玩家周围更亮，远处稍暗，营造氛围但不遮挡视野
- Max 在远处会变暗淡，靠近时恢复正常

### 3.5 游戏流程
- 开始画面：标题 + "Press SPACE to Start"
- 游戏开始：玩家出生在客厅，Max 在远处书房
- 生存计时：显示存活时间
- 3 条命耗尽 → 死亡画面：显示存活时长，可按 SPACE 重新开始
- 右上角实时显示剩余爱心（♥♡）

### 3.6 氛围效果
- 暗色调配色
- 浮尘粒子漂浮
- Max 靠近时画面边缘变红
- 屏幕震动（Max 很近时）
- 心跳脉冲视觉效果（UI 缩放 + 计时器变红）
- 威胁方向指示器（红色箭头指向 Max 方位）

### 3.7 道具系统
道具随机散落在地图上，拾取后立即生效，8–12 秒后在别处重新生成：
- **速度提升（SPD / 黄色）**：玩家速度从 4.5 提升到 7.0，持续 5 秒
- **隐身（INV / 蓝色）**：Max 无法追踪，随机游荡，持续 6 秒，玩家半透明
- **刀（KNIFE / 红色）**：拾取后保留，下次被 Max 碰到时自动使用，击晕 Max 3 秒
